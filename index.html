Vaccine Variancy

Welcome to the SQL Saviours Vaccine Variancy WebApp. Here you can access a variety of visualizations compiled via embedded Tableau Dashboards using the Javascript API. Due to the Javascript API, all nodes in the Dashboard are interactive and will display more information. First- Cases and Vaccinations, Second- IL vs. IA, Third- Hesitancy and SVI
